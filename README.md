# Inventory Management System (PHP OOP)

A simple yet powerful Inventory Management System built with **PHP** using **Object-Oriented Programming (OOP)** principles. This project is designed to help businesses efficiently manage their products, categories, and stock levels with secure user authentication and CRUD functionality.

## Features
- 🔐 User Authentication (Register, Login, Logout)
- 📦 Product Management (Add, Edit, Delete, View)
- 🗂️ Category Management
- 📊 Stock Tracking & Updates
- 👤 User-Friendly Dashboard
- ⚙️ CRUD Operations for seamless data handling
- 🛠️ Scalable and maintainable codebase

## Technologies Used
- PHP (OOP)
- MySQL (Database)
- HTML, CSS, JavaScript (Frontend)
- Bootstrap (UI Styling)
